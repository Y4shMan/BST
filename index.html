Tree Traversals

How to use:

• Insert: Enter a number and click Insert to add it to the tree
• Delete: Enter a number and click Delete to remove it from the tree
• Traversals: Click any traversal button to see the order of nodes
• Clear: Click Clear Tree to remove all nodes
